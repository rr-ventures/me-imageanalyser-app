# Prompts

This folder contains prompt assets used to research, audit, and run the app.

## Contents

- `MASTER_REPO_QA_PROMPT.md` — master Cursor prompt for deep repo QA/product/scale review
- `GEMINI_RESEARCH_PROMPT.md` — research prompt for Lightroom preset evidence gathering
- `CONDENSE_REPORTS_PROMPT.md` — prompt for merging research reports into a YAML artifact
- `CONDENSE_CROP_RESEARCH_PROMPT.md` — prompt for turning crop research into a YAML artifact
- `metadata_prompt.py` — runtime Gemini metadata extraction prompt used by the backend

`library/` now stores app data artifacts only, not authoring prompts.
